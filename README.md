A real-time lab availability scanner for BRACU CSE students based on the Spring 2026 class routine. Users can select a day and time slot to instantly view available lab rooms.



🚀 Feature:

--> 🔍 Filter labs by Day

--> ⏰ Filter labs by Time Slot (8 AM / 11 AM / 2 PM)

--> 📊 Dynamic availability calculation from schedule data

--> 🖥️ Responsive UI

--> ✨ Animated cyber-retro-style interface

--> ⚡ Instant results without page reload



🛠️ Tech Stack:

--> HTML5 — Structure

--> CSS3 — Styling, animations, responsive design

--> Vanilla JavaScript (ES6) — Filtering logic & DOM manipulation

--> No frameworks. No backend. Pure frontend implementation.



🎯 Project Type

Frontend Web Application
Static Single-Page Application (SPA)
